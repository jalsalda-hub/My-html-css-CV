# Stack Tecnológico

## Frontend
- **HTML5:** Estructura semántica y accesible.
- **CSS3:** Estilizado moderno sin frameworks pesados.
    - Uso de **CSS Variables (Custom Properties)** para temas y mantenibilidad.
    - **Flexbox** y **CSS Grid** para layouts.
    - Metodología BEM (Block Element Modifier) para nomenclatura de clases.
- **JavaScript (Vanilla ES6+):** Lógica de interacción, manejo del DOM, internacionalización y cambio de temas.

## Herramientas de Desarrollo y Diseño
- **Control de Versiones:** Git & GitHub.
- **Iconografía:** FontAwesome o SVG directos (optimizado).
- **Fuentes:** Google Fonts (seleccionadas en Sistema de Diseño).

## Infraestructura
- **Hosting:** GitHub Pages.
- **Dominio:** (Por definir/configurar).
