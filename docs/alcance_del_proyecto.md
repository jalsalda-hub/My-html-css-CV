# Alcance del Proyecto

## Objetivo General
Construir un portafolio profesional para vender una marca personal como Desarrollador FullStack experto en Procesos Industriales, Manufactura, Supply Chain, Logística, Ecommerce y Marketing Digital.

## Público Objetivo
- **Principal:** Clientes potenciales para Freelance y Consultoría.
- **Secundario:** Reclutadores y contactos profesionales.

## Alcance Funcional
El sitio será un Portafolio Web con las siguientes características clave:

1.  **SPA (Single Page Application) Feel:** Navegación fluida en una sola página con secciones claramente definidas (Inicio, Sobre Mí, Servicios, Proyectos, Blog, Contacto).
2.  **Multilingüe:** Soporte completo para Español e Inglés con cambio instantáneo.
3.  **Diseño Industrial/Tech:** Estética profesional que refleje la solidez de la ingeniería industrial y la innovación tecnológica.
4.  **Modo Oscuro/Claro:** Preferencia de usuario respetada y persistente.
5.  **Casos de Estudio Detallados:** Secciones específicas para profundizar en proyectos clave con enfoque en valor de negocio (Business Value).
6.  **Blog Técnico:** Sección para demostrar autoridad en la materia.
7.  **Descarga de CV:** Opción accesible para descargar el CV en formato PDF.
8.  **Formulario de Contacto Funcional:** Integración para recepción de mensajes directos.

## Exclusiones (Out of Scope)
- Backend complejo (se utilizarán servicios de terceros para formularios si es necesario, o `mailto` mejorado).
- CMS autohospedado (el contenido del blog será estático o MD gestionado).
